# ❤️ Love Lexicon Analysis Using NLP

## Project Overview
This project explores how love is expressed in song lyrics using Natural Language Processing (NLP). More than 57,000 songs were analyzed to identify different categories of love-related words and measure the intensity of love expressions.

## Objectives
- Clean and preprocess song lyrics
- Build a love lexicon
- Calculate Love Scores
- Analyze artists and songs
- Visualize insights

## Technologies Used
- Python
- Pandas
- NLTK
- Matplotlib
- Plotly
- Jupyter Notebook

## Files
- data_exploration.ipynb
- text_processing.ipynb
- love_lexicon.ipynb

## Results
This project analyzes more than 57,000 song lyrics to understand how love is expressed in music. The analysis includes:

- Top 10 songs with the highest love scores
- Top artists using love-related words
- Distribution of love scores
- Love category distribution
- Most common love-related categories
- Love level distribution
