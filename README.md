# ❤️ Love Lexicon Analysis Using NLP

> An NLP project that analyzes expressions of love in over **57,000 Spotify song lyrics** using text preprocessing, a custom love lexicon, and data visualization.

---

## 📖 Project Overview

This project explores how love is expressed in song lyrics using **Natural Language Processing (NLP)** techniques. A custom love lexicon was created to measure the presence of love-related words in each song. The project then analyzes artists, songs, and lyrical patterns through visualizations.

---

## 🎯 Objectives

- Clean and preprocess song lyrics
- Build a custom love lexicon
- Calculate love scores for songs
- Analyze artists and songs
- Visualize love-related insights

---

## 🛠️ Technologies Used

## 📌 Key Findings

- Over 57,000 Spotify songs were analyzed.
- Love-related words appear frequently across multiple music genres.
- Some artists consistently use significantly more love-related vocabulary.
- The custom love lexicon provides a simple and interpretable method for scoring lyrics.
